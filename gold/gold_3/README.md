## 이 단계에서 익힌 핵심 패턴
- BFS에서 최단 거리 계산 후 parent 배열로 경로를 역추적
- 방문 여부와 부모 정보를 하나의 배열로 관리
- 경로 리스트를 큐에 저장하면 시간 초과가 발생함을 경험

## 자주 한 실수 & 배운 점
- `parent[start]`를 `-1` 또는 `-2`로 두면 음수 인덱싱으로 무한 루프 발생
- `list.reverse()`의 반환값이 `None`이라는 점을 놓쳐 경로 출력 오류 발생
- BFS 큐에는 정수(노드)만 저장해야 함
